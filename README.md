# smartBreeder_test

## Start

- Clone repo: git clone https://github.com/leoccomp/smartBreeder_test.git
- Acessar diretório: cd smartBreeder_test/frontend/
- Instalar dependências: yarn (ou npm)
- Executar: yarn start (ou npm run start)
